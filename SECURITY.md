# Security Policy

## Supported Versions

This is a template repository. Security fixes are applied to the latest version on the `main` branch only.

## Reporting a Vulnerability

Please **do not** open a public GitHub issue for security vulnerabilities.

Report vulnerabilities by email to **abhinavkumarsintoo@gmail.com** with:

- A description of the vulnerability
- Steps to reproduce
- Potential impact
- Any suggested fix (optional)

You will receive a response within 72 hours. If the report is confirmed, a fix will be prioritised and released as soon as possible.
