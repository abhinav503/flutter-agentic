## Summary

Adds a second real reference app — a streaming AI chat — and the reusable streaming building blocks behind it, alongside doc cleanup and a stronger review checklist.

---

## Features

- A new streaming AI chat app: replies stream in token by token, with a one-shot mode you can toggle, markdown-rendered answers, and the ability to stop a reply mid-stream or retry after an error
- Chat works out of the box with a built-in local mock — no account or key needed — and switches to a real model the moment you paste an API key in the app
- New shared UI building blocks for any app: a themed dropdown menu, an inline "working…" dots indicator, and a compact density option for text fields
- Per-app feature guides so each app documents what it actually does

## Agent Context Improvements

- A new guide for streaming responses (server-sent events and token streams) through the architecture, so agents follow one consistent pattern for real-time data
- The code review checklist now explicitly checks for reusing existing shared components and promoting genuinely generic code into the shared package, instead of reinventing it
- Architecture and convention docs updated to list the new shared components and to remove references to networking tooling that is no longer used
