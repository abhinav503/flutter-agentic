## Summary

Adds push notifications to the template, ships the skills as an installable Claude Code plugin, and brings tooling to run the apps on real mobile devices.

---

## Features

- Push notifications in the reference app: receive messages whether the app is open, backgrounded, or closed; tap a notification to open a specific page; and show rich notifications with images.
- The template's skills are now installable as a Claude Code plugin — add it to any Flutter project with two commands, no fork required (this release ships the feature-scaffolding skill as a pilot).
- Run the template's apps on a connected mobile device, with a guided local-setup flow for first-time machine setup.
- The in-browser terminal app can now detect connected devices, arrange app windows side by side, and walk through local environment setup.

---

## Agent Context Improvements

- A new skill that adds Firebase push notifications to an app, organized as shared app code plus separate iOS and Android setup tracks, including guidance to test delivery on a real device rather than an emulator.
- A new skill that connects an app to Firebase end to end, handling the command-line setup flow and platform configuration.
- Agent instructions across all supported tools updated so every assistant knows the new Firebase and notification skills.
